Course Objectives

The objective of this course is to expose students to methodology, techniques, design, and testing interactive systems. The course focuses on the construction of user interfaces.

Approach:

  • Software architecture
  • Build a few interactive applications
  • Heavy in OO and application frameworks

Technotes

Technote are individual assignments. The objective is:

  • to give you practice preparing a description and a brief demo of a new technology
  • to give you practice reviewing each other's work
  • More about TechNotes in a moment

Programming Projects

There will be four programming projects, some individual and some in small groups. The same group is expected to work together throughout the semester. Students self select themselves into groups; the professor can assist if you don’t know others in the class.

All programming assignments are in Java. All involve a starting code set and you modifying the code.
